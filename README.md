# 京剧宣传管理系统 java166

## 项目概述

京剧宣传管理系统是一款结合了现代互联网技术与传统京剧文化的平台。它使用Spring Boot与Vue3技术栈，旨在通过丰富的功能模块加强京剧的宣传与推广，提高用户对京剧文化的学习与赏析兴趣。

## 技术栈

- 后端：Spring Boot 3, MyBatis
- 数据库：MySQL
- 前端：Vue3, Element Plus

## 功能模块

### 前台功能

- **京剧展示**：展示经典剧目、历史发展、艺术特色等内容。
- **交流论坛**：提供用户互动交流的平台，分享京剧学习心得。
- **知识趣答**：通过趣味问答环节，普及和加深京剧知识。
- **积分商城**：用户通过参与活动获取积分，可兑换文创产品。
- **宣传栏板**：发布和展示各类宣传信息。

### 后台功能

- **内容管理**：实现对展示内容的添加、修改、删除操作。
- **活动管理**：发布和管理各类互动活动。
- **用户管理**：管理注册用户的信息。
- **系统设置**：进行系统参数配置。

## 系统角色

系统包含两类角色：前台用户和后台管理员。

- **前台用户**：浏览内容，参与交流与活动，兑换积分商品。
- **后台管理员**：管理内容、活动、用户及系统设置。

## 运行环境

- **后端**：Java 17+环境，MySQL数据库
- **前端**：现代浏览器支持

## 部署步骤

1. 准备后端所需环境，配置数据库。
2. 导入项目源码至开发工具。
3. 部署前端项目至Web服务器。
4. 运行后端项目，确保前后端通信正常。
